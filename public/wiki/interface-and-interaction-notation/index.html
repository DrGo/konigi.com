Interface and Interaction Notation

About

Resources for annotating interfaces and interaction. Includes notation techniques for systems using input devices or gestural recognition for interpreting human body movement and other attribute recognition for input.

General

  • Interactive Sketching Notation. Jakub Linowski’s techniques for specifying interaction. See Video.
  • A shorthand for designing UI flows. Ryan Singer’s technique for illustrating flow.
  • Visual Vocabulary. Jesse James Garret’s system for diagramming information architecture and interaction design.
  • Wireframes Magazine. See examples of Annotation and User Flow.
