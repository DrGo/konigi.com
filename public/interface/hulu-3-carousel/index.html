Hulu 3 Up Carousel

Hulu's new 3-up carousel is being utilized in the site's channels sections to offer a promotional space that takes up less screen height than their 1 up carousel, allowing the display of 3 promo items. This is very similar to the horizontal scrolling Cover Flow module in use on Apple.com interior pages.

1. First set of channels. Scroll bar below allows the user to view more.
2. Second set of channels. Hovering over a channel reveals it's name and an "explore" button to jump to it.
